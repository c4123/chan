Detailed Description

Map-reduce framework implemented using Chan.

map() example

Take an array of 8-bit RGB tuples from a color image and map to convert to grayscale, outputing an array of doubles.

The function to be applied is defined: 

    int grayscale(void *_dst, void *_src)
    { double        *dst = (double*) _dst;
      unsigned char *src = (unsigned char*) _src;
      *dst = 0.630*src[0] + 0.310*src[1] + 0.155*src[2];
      return 0;
    }

Perform the grayscale conversion in parallel over a 512x512 RGB image. The output image will be dynamically allocated. 

    MRData result = 
      map(MREmpty(sizeof(double)),
          MRPackage(image,3,512*512*3),
          grayscale);
    // use the result
    MRRelease(&result);

If the output image has already been allocated: 

    double out[512*512];
    map(MRPackage(out,8,512*512)),
        MRPackage(image,3,512*512*3),
        grayscale);
    // use "out"

static int maybe_alloc_dst ( MRData *  dst,
MRData *  src 
) [static]

Allocates dst data if necessary, otherwise shrinks src to fit.

There are four cases:

  • dst is ill defined (bytesof_elem is 0)
  • dst describes a buffer bigger than *src
  • dst describes a buffer smaller than *src
  • dst is "empty"

The first case is treated as an error, so the function returns 0.

The second case requires no work. Everything is ok, so the function returns 1.

The third case is treated as an error, so the function returns 0.

The last case requires the appropriate space is allocated. The number of output elements is computed from src. This is then multiplied by the dst bytesof_elem to determine the final buffer size to request from the heap.

Parameters:
src Description of the source buffer.
dst Description of the destination buffer.
Returns:
1 on success, otherwise 0
